# 🚀 Decode Labs - Project 3: Interactive Web Elements

## 📌 Project Overview

This project demonstrates the fundamentals of JavaScript by creating interactive web elements with a modern and responsive user interface. It combines HTML, CSS, and JavaScript to deliver a smooth and engaging user experience.

## ✨ Features

- 🌞 Dynamic Greeting (Morning / Afternoon / Evening)
- 🕒 Live Date & Time Display
- 💬 Random Motivational Message Generator
- 🌙 Dark / Light Mode Toggle
- 🔢 Interactive Counter (Increase, Decrease & Reset)
- ❤️ Like Counter
- 👀 Show / Hide Information Section
- 🔔 Toast Notifications
- 🎨 Modern Glassmorphism User Interface
- ✨ Smooth Hover Effects & Animations
- 📱 Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
